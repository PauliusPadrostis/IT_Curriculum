## Context

Key alignment tasks; students must explain how join errors change conclusions.\
\

## Completion Definition

Student completes a realistic workflow:

- join datasets

- resolve mismatch problems

- produce a basic summary (pivot-like or grouped summary)

- explain how mismatches impacted results

## Minimum Evidence

Submission includes:

- joined dataset output (or clear join result)

- mismatch resolution notes

- summary output (table)

- 3–5 sentence explanation of impact

## Cognitive Level (Bloom's)

**Bloom:** Analyze

## Readiness Gate

✔ All true:

- mismatch resolution changes are correct

- summary output matches the corrected data

- impact explanation is causal and specific\
  If not → not ready for P2/A2.
