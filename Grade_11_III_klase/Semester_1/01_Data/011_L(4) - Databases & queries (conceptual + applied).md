## Context

**Grade 11 anchor:** “duomenų bazės ir užklausos.”\
\

## Sub-topics

- Table/record/field; primary key; relationships (basic)

- Query logic: selection (WHERE), sorting (ORDER), grouping/aggregation (conceptual)

- Reading query results and explaining what a query does

- Common mistakes: wrong condition, wrong join assumption, aggregation misuse

## Completion Definition

Student can:

- Explain what a database/table represents (records/fields, keys)

- Interpret what a query is doing (selection, filtering, projection, sorting, basic grouping where applicable)

- Predict results of a query on a small dataset

- Write or choose the correct query (SQL-like or query-builder logic) for a specified information need

Focus: reasoning + correctness, not memorizing syntax.

## Minimum Evidence

Student independently completes:

- ≥2 “predict the result set” tasks (given a table + query)

- ≥2 “choose/write the correct query” tasks for a stated question

- ≥1 task identifying why a query is wrong (wrong filter, missing condition, wrong field)

Responses must include:

- the correct result (or correct query)

- 1–2 sentence justification for at least one item

## Cognitive Level (Bloom's)

**Bloom:** Understand → Analyze

## Readiness Gate

✔ All true:

- Correctly distinguishes field vs record vs key

- Can predict query outcome on small data

- Can choose/write a query matching intent\
  If not → not ready for Workflow 3.
