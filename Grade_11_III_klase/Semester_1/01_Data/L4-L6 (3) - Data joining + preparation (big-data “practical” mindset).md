**Semester:** Semester 1
**Status:** 🚧 WIP

## Context

**Grade 11 anchor:** “didelių duomenų jungimas ir paruošimas analizei.”\
\

## Sub-topics

- Joining datasets (keys, mismatches, duplicates)

- Cleaning: missing values, inconsistent categories, type conversion

- “Analysis-ready table” criteria

## Completion Definition

Student can explain and perform correct joining/prep actions:

- identify join keys

- detect mismatches (missing keys, duplicates, inconsistent categories)

- justify preparation choices (filtering, column normalization)

## Minimum Evidence

Student submits:

- two small datasets

- chosen join key(s) with justification

- joined output (or join plan if tool-limited)

- list of ≥2 mismatch issues found and how handled

## Cognitive Level (Bloom's)

**Bloom:** Apply → Analyze

## Readiness Gate

✔ All true:

- join key choice is valid

- mismatches identified (not ignored)

- preparation steps justified\
  If not → not ready for Workflow 2.
