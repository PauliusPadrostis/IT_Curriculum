## Completion Definition

Student independently completes an analytical assessment that demonstrates:

- correct query reasoning (choose/correct a query)

- correct interpretation of returned results

- correct limitation recognition

- concise short-answer justification

## Minimum Evidence

Assessment includes:

- ≥1 “choose the correct query” item

- ≥1 “correct the flawed query/logic” item

- ≥1 “interpret results” item requiring:

  - a numeric reference

  - a limitation/caution statement

Student outputs must be short-answer and evidence-based.

## Cognitive Level (Bloom's)

**Bloom:** Analyze → Evaluate

## Readiness Gate

✔ All true:

- Query logic matches intent

- Interpretation uses at least one correct numeric reference

- Includes a valid limitation

- No unsupported inference\
  If not → remediation required before Mini-block 4 (presentation + ethics/AI lens).
