## Context

**Workflow**

- Given dataset → choose representation → design visual layout

- Students must justify visual choices in writing (short paragraph).\
  \

## Completion Definition

Student designs a structured poster that:

  - Uses vector layout logic

  - Includes one data visualization

  - Contains concise narrative with ≥1 numeric reference

  - Applies ethical presentation principles

## Minimum Evidence

Submission includes:

  - Poster file

  - Underlying data source

  - Short design rationale (4–6 sentences)

## Cognitive Level (Bloom's)

**Bloom:** Apply → Analyze

## Readiness Gate

✔ Data accurate\
  ✔ Layout structured\
  ✔ Numeric reference included\
  ✔ No misleading design element
