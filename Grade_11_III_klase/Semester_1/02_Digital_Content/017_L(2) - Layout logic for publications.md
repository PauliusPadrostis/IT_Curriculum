# L17 (2) — Layout logic for publications

**Type:** 📖 Lesson
**Grade:** Grade 11 (III gimnazijos klasė)
**Module:** Digital Content – Vector Graphics & Publications
**Semester:** Semester 1
**Sequence:** 17 of grade year
**Lessons allocated:** 2
**Status:** 🚧 WIP

---

## Context

**Emphasis:** Design decisions must be justified.\
\

## Sub-topics

- Visual hierarchy (title → subtitle → body → data)

- Grid systems and alignment discipline

- Typography pairing basics

- White space as structure

## Completion Definition

Student can:

- Apply hierarchy (title, subtitle, body)

- Use grid/column structure

- Control whitespace intentionally

- Avoid clutter and random placement

## Minimum Evidence

Student produces:

- One layout draft using grid structure

- Clear visual hierarchy (size/weight/contrast)

- Short explanation (3–5 sentences) of layout choices.

## Cognitive Level (Bloom's)

**Bloom:** Apply → Analyze

## Readiness Gate

✔ All true:

- Clear hierarchy visible

- Grid logic evident

- No arbitrary spacing\
  If not → not ready for data visualization principles.
