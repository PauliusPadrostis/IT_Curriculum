# L16 (2) — Vector graphics foundations (publication mindset)

**Type:** 📖 Lesson
**Grade:** Grade 11 (III gimnazijos klasė)
**Module:** Digital Content – Vector Graphics & Publications
**Semester:** Semester 1
**Sequence:** 16 of grade year
**Lessons allocated:** 2
**Status:** 🚧 WIP

---

## Context

**Goal:** Students understand *why* vector is used in formal publications.\
\

## Sub-topics

- Vector vs raster (precision use-case reasoning)

- Formats (SVG, PDF, etc.) and when to use them

- Resolution vs scaling logic

- Transparency and layering

## Completion Definition

Student can:

- Explain the difference between vector and raster graphics

- Use vector tools to create clean shapes, aligned layouts, and scalable graphics

- Apply basic alignment, grouping, layering, and spacing principles

- Produce a simple publication-ready visual (poster-style layout base)

Focus: precision + scalability + clean structure.

## Minimum Evidence

Student submits:

- One vector-based layout containing:

  - ≥3 aligned elements

  - correct use of grouping/layers

  - consistent spacing

- Short note (3–4 sentences) explaining why vector is appropriate for publication.

## Cognitive Level (Bloom's)

**Bloom:** Apply → Understand

## Readiness Gate

✔ All true:

- Elements aligned (no accidental misplacement)

- Uses vector objects (not embedded raster screenshot)

- Explains vector advantage correctly\
  If not → not ready for layout logic.
