# L5–L6 (2) — Data visualization principles (design side, not spreadsheet mechanics)

**Type:** 📖 Lesson
**Grade:** Grade 11 (III gimnazijos klasė)
**Module:** Digital Content – Vector Graphics & Publications
**Semester:** Semester 1
**Status:** 🚧 WIP

---

## Context

**Supports Data block reasoning without duplicating it.**\
\

## Sub-topics

- Choosing chart type by question

- Avoiding misleading visuals

- Color meaning and accessibility (contrast)

- Legend clarity and labeling discipline

## Completion Definition

Student can:

- Explain why certain chart types communicate specific relationships

- Apply visual clarity rules:

  - no distortion

  - consistent color logic

  - readable labels

- Identify misleading design choices

This is visual logic, not spreadsheet mechanics.

## Minimum Evidence

Student submits:

- One redesigned chart (corrected from a flawed version)

- Short justification including:

  - what was misleading

  - how it was corrected

## Cognitive Level (Bloom's)

**Bloom:** Analyze

## Readiness Gate

✔ All true:

- Correct chart type selected

- Identifies at least one real misleading element

- Redesign improves clarity\
  If not → not ready for data-driven compositions.
