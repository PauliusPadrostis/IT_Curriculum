## Context

**Bridge:** Data analysis → visual communication.\
\

## Sub-topics

- Turning computed data into visual story

- Integrating icons, vector elements, and charts

- Maintaining consistency across visuals

- Common infographic failures

## Completion Definition

Student can integrate:

- data content

- visual hierarchy

- chart logic

- minimal narrative

Into a single coherent visual piece.

## Minimum Evidence

Student produces:

- One data-backed visual layout

- Includes:

  - one chart

  - one numeric reference in text

  - consistent color logic

- No decorative clutter.

## Cognitive Level (Bloom's)

**Bloom:** Apply → Analyze

## Readiness Gate

✔ All true:

- Chart appropriate

- Numeric reference correct

- Layout coherent and readable\
  If not → not ready for ethics layer.
