## Context

## 1️⃣ Completion Definition (Observable Capability)

Student independently completes an assessment set that demonstrates:

- Correct reasoning about arrays/strings/sort/search behavior

- Ability to trace execution and predict outputs/state

- Ability to identify algorithm preconditions (sorted vs unsorted)

- Ability to diagnose a flaw category (bounds, swap logic, accumulator init, condition placement)

- Ability to write **one short code fragment** (10–25 lines) that implements a specified sub-task correctly

The short code component must be aligned with the reasoning portion (same domain).
