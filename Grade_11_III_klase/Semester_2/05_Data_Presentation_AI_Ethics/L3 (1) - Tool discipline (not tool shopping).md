# L3 (1) — Tool discipline (not tool shopping)

**Type:** 📖 Lesson
**Grade:** Grade 11 (III gimnazijos klasė)
**Module:** Data – Presentation, AI & Ethics
**Semester:** Semester 2
**Status:** 🚧 WIP

---

## Context

Versioning basics (document history), naming conventions, folder structure, access rights.\
\

## Completion Definition

Student can explain why tool discipline matters (consistency, traceability, reduced fragmentation) and can select a minimal toolset aligned to workflow requirements rather than novelty.

## Minimum Evidence

Given a project description, student selects a minimal set of tools (e.g., communication, task tracking, document storage) and states clear usage rules (where decisions go, where files live, how naming/versioning works).

## Cognitive Level (Bloom's)

Apply → Evaluate

## Readiness Gate

Do the rules prevent fragmentation (single source of truth) and define where key artifacts live? Yes/No
