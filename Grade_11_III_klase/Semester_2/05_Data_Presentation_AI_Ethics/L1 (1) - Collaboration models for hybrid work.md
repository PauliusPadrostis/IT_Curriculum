# L1 (1) — Collaboration models for hybrid work

**Type:** 📖 Lesson
**Grade:** Grade 11 (III gimnazijos klasė)
**Module:** Data – Presentation, AI & Ethics
**Semester:** Semester 2
**Status:** 🚧 WIP

---

## Context

Sync vs async; meeting vs doc-driven; choosing modes by goal.\
\

## Completion Definition

Student can compare common collaboration models used in hybrid work (synchronous vs asynchronous, centralized vs distributed coordination, meeting-driven vs document-driven workflows) and select an appropriate model for a given team situation.

## Minimum Evidence

Given 4 team scenarios, student selects a collaboration model for each and justifies the choice using at least two constraints (time zones, task interdependence, urgency, documentation needs).

## Cognitive Level (Bloom's)

Understand → Analyze

## Readiness Gate

Can the student justify a model choice based on scenario constraints rather than preferences (“I like meetings”)? Yes/No
