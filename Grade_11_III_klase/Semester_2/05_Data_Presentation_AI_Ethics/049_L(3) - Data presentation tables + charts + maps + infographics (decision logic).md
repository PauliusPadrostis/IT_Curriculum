## Context

**Grade 11 anchor:** “duomenų pateikimas (lentelės, diagramos, žemėlapiai, infografikai).”\
\

## Sub-topics

- Choosing representation by question type

- Chart misuse and misleading visuals

- Map use cases (when geography matters, when it doesn’t)

- Infographic structure: message hierarchy (title → key metric → evidence)

## Completion Definition

Student can:

- Choose an appropriate presentation format (table/chart/map/infographic) based on the reporting question and data type

- Justify the choice (what it reveals; what it hides)

- Apply basic correctness rules:

  - correct labels/units

  - readable scales

  - appropriate categories/aggregation

- Avoid misleading representations (truncated axes without justification, wrong chart type, clutter)

## Minimum Evidence

Student submits:

- 2 presentation choices for 2 different reporting questions, each including:

  - chosen format

  - the produced visual/table (or structured plan if tool-limited)

  - 2–3 sentence justification

- At least one item must include a numeric reference in the justification (“shows X is higher than Y by …”)

## Cognitive Level (Bloom's)

**Bloom:** Analyze → Evaluate

## Readiness Gate

✔ All true:

- Format matches intent (trend/comparison/composition/distribution/location)

- Visual/table has correct labels/units and readable scale

- Justification references evidence (not opinion)\
  If not → not ready for Workflow 4.
