# L4 (1) — Domain, URL anatomy, DNS resolution path

**Type:** 📖 Lesson
**Grade:** Grade 11 (III gimnazijos klasė)
**Module:** Data – Presentation, AI & Ethics
**Semester:** Semester 2
**Status:** 🚧 WIP

---

## Context

URL components; domain hierarchy; what DNS actually translates.\
\

## Completion Definition

Student can break down a URL into components (protocol, domain, subdomain, path, parameters) and describe the high-level DNS resolution path from domain name to IP address.

## Minimum Evidence

Given 3 URLs, student correctly labels structural components and writes a stepwise explanation of how DNS resolves the domain to an IP (client → resolver → DNS hierarchy → response).

## Cognitive Level (Bloom's)

Understand → Analyze

## Readiness Gate

Can the student describe resolution as a process flow rather than a single-step lookup? Yes/No
