# P3 (1) — Short argument responses (ethics/regulation)

**Type:** 🏃 Practice
**Grade:** Grade 11 (III gimnazijos klasė)
**Module:** Data – Presentation, AI & Ethics
**Semester:** Semester 2
**Status:** 🚧 WIP

---

## Context

2–3 sentence justification tasks (why a use case is risky/acceptable).\
\

## Completion Definition

Student can write short, exam-style arguments that:

- make a clear claim

- support it with one concrete example or consequence

- include one limitation/counterpoint

- stay concise (not essay)

## Minimum Evidence

Timed writing pack includes:

- ≥2 prompts

- each response 6–10 sentences

- includes:

  - one concrete example

  - one counterpoint/limitation

  - one ethics/regulation concept used correctly

## Cognitive Level (Bloom's)

**Bloom:** Evaluate

## Readiness Gate

✔ All true:

- Uses at least one correct ethics/regulation concept

- Includes counterpoint/limitation (not one-sided)

- Argument is coherent and concise\
  If not → not ready for A4.
