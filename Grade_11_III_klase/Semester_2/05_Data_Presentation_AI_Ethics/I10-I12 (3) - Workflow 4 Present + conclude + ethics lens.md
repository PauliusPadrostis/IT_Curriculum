# I10–I12 (3) — Workflow 4: Present + conclude + ethics lens

**Type:** 🔗 Integration
**Grade:** Grade 11 (III gimnazijos klasė)
**Module:** Data – Presentation, AI & Ethics
**Semester:** Semester 2
**Status:** 🚧 WIP

---

## Context

- One dataset → appropriate visualization → short conclusion
- Add AI/ethics angle: “what could go wrong if used for decisions?”\
  \

## Completion Definition

Student can complete an end-to-end reporting workflow:
  - select an appropriate presentation form

  - produce the presentation correctly

  - write a concise conclusion grounded in data

  - include an ethics lens: what could mislead, what is missing, what should be disclosed

## Minimum Evidence

Student submits a mini-report containing:
  - one visual/table/map/infographic

  - the underlying data range or extracted subset used

  - a written conclusion (4–6 sentences) including:

    - one quantified finding (number/%/difference)

    - one limitation/caution

    - one ethics note (risk of misinterpretation, bias, missing data, privacy sensitivity)

  No sentence starters. No scaffold frames.

## Cognitive Level (Bloom's)

**Bloom:** Analyze → Evaluate

## Readiness Gate

✔ All true:
  - Conclusion contains ≥1 numeric reference

  - No unsupported inference (no causation without evidence)

  - Includes a meaningful limitation + ethics note\
    If not → not ready for AI/ethics analysis block (L14–L16).
