## 1️⃣ Completion Definition

Student independently demonstrates practical coordination competence by organizing a mini-workflow: assign roles, define a workflow contract, enforce tool discipline, and handle a feedback/conflict prompt.

## 2️⃣ Minimum Evidence

Assessed submission includes:\
A coherent workflow contract\
Correct role/responsibility mapping\
Tool-use rules that ensure traceability\
A written response to a feedback/conflict scenario that is specific and actionable

## 3️⃣ Bloom's Taxonomy Level

Analyze → Evaluate

## 4️⃣ Readiness Gate

Does the student produce a coherent, enforceable coordination system (not just advice) and apply it correctly to the given scenario? Yes/No
