# TL03 — Network Concepts

**NŠA requirement:** Kompiuterių tinklai
**Delivery:** Homework reading + self-check quiz
**Estimated reading time:** 25 min

---

## Key Concepts Students Must Know

- TCP/IP model layers and their functions (not memorization — reasoning tool)
- IP addressing basics (IPv4, subnet concept)
- Common protocols and their port numbers (HTTP/80, HTTPS/443, FTP/21, SMTP/25)
- Domain name system: URL anatomy, DNS resolution path
- How data flows from browser to server and back

## Exam Relevance

2–3 questions in VBE Part 1 on network layers, protocols, and DNS resolution.
