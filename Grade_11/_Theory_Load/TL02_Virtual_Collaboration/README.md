**NŠA requirement:** Virtualaus bendravimo/bendradarbiavimo priemonės
**Delivery:** Homework reading + self-check quiz

## Key Concepts Students Must Know

- Synchronous vs asynchronous communication tools
- Collaboration platforms (Google Workspace, MS Teams, Slack — awareness level)
- Hybrid work models and their communication requirements
- Roles and responsibilities in virtual teams
- Tool discipline: choosing the right tool for the task

## Exam Relevance

1–2 questions in VBE Part 1 on collaboration tool selection and virtual team scenarios.
