# TL01 — Cryptography Basics

**NŠA requirement:** Kriptografinės sistemos, viešasis ir privatusis raktas
**Delivery:** Homework reading + self-check quiz
**Estimated reading time:** 20 min

---

## Key Concepts Students Must Know

- Symmetric encryption: one key for both encrypt and decrypt (AES)
- Asymmetric encryption: public key encrypts, private key decrypts (RSA)
- Key distribution problem and why asymmetric encryption solves it
- Digital certificates and certificate authorities
- What the browser padlock means

## Exam Relevance

1–2 multiple choice questions in VBE Part 1 on encryption types and certificate concepts.
