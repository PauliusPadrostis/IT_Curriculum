**Year focus:** Intermediate data literacy and programming — database queries, visual communication, string and array manipulation, network fundamentals, and a VBE diagnostic to benchmark exam readiness.

## Progression logic

Semester 1 consolidates data and communication skills; Semester 2 extends programming to collections and file I/O, then broadens to literacy topics (AI ethics, safety, networks) before a diagnostic assessment that maps readiness for Grade 12 exam prep.

Modules 06, 07, and 08 deliver their L-type lessons as **Theory Load** homework packages — only I and A lessons are scheduled classroom sessions. Module 10 provides the in-class assessment for all theory-loaded content.

## Assessment points

- Practical data workflow assessment (Module 01)
- Practical assessments in programming modules (03, 04)
- Theory quizzes for theory-loaded modules (Module 10)
- VBE-format diagnostic (Module 09)
