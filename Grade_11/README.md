# Grade 11 – Data · Digital Content · Arrays · AI Ethics · Networks

**Year focus:** Intermediate data literacy and programming — database queries, visual communication, string and array manipulation, network fundamentals, and a VBE diagnostic to benchmark exam readiness.

## Modules

| Semester | Module | Topic |
|----------|--------|-------|
| 1 | [01_Data](Semester_1/01_Data/) | Database concepts, SQL-style queries, joins, aggregation |
| 1 | [02_Digital_Content](Semester_1/02_Digital_Content/) | Infographics, data visualisation, visual design principles |
| 2 | [03_Programming_Strings_Arrays](Semester_2/03_Programming_Strings_Arrays/) | String operations, 1D arrays, traversal and aggregation patterns |
| 2 | [04_Programming_Insert_Delete](Semester_2/04_Programming_Insert_Delete/) | Array modification: insert, delete, update; file I/O pipeline |
| 2 | [05_Data_Presentation_AI_Ethics](Semester_2/05_Data_Presentation_AI_Ethics/) | Presenting findings, AI-generated content ethics |
| 2 | [06_Safety](Semester_2/06_Safety/) | Cybersecurity, safe online behaviour, data protection |
| 2 | [07_Virtual_Communications](Semester_2/07_Virtual_Communications/) | Professional digital communication, netiquette |
| 2 | [08_Networks](Semester_2/08_Networks/) | Network types, protocols, IP addressing basics |
| 2 | [09_VBE_Diagnostic](Semester_2/09_VBE_Diagnostic/) | Full VBE-format diagnostic: identify gaps before Grade 12 |

## Progression logic

Semester 1 consolidates data and communication skills; Semester 2 extends programming to collections and file I/O, then broadens to literacy topics (AI ethics, safety, networks) before a diagnostic assessment that maps readiness for Grade 12 exam prep.

## Assessment points

- Practical data workflow assessment (Module 01)
- Practical assessments in programming modules (03, 04)
- VBE-format diagnostic (Module 09)
