## Overview

- Create a vector-based visual publication using provided data

- Criteria:

  - Correct chart choice

  - Visual hierarchy

  - Clarity and consistency

  - Proper format/export.\
    \

---

## 1️⃣ Completion Definition

Student independently produces a final design task that integrates:

    - Vector layout logic

    - Data visualization

    - Clear narrative

    - Ethical presentation discipline

## 2️⃣ Minimum Evidence

Submission includes:

    - Final design

    - Data source

    - Short written justification including:

      - One numeric reference

      - One ethical design consideration

## 3️⃣ Bloom's Taxonomy Level

**Bloom:** Apply → Evaluate

## 4️⃣ Readiness Gate

✔ Data accurate\
    ✔ Layout structured\
    ✔ Numeric reference present\
    ✔ No misleading representation\
    ✔ Justification coherent
