# Grade 11 – Semester 1

**Semester goal:** Develop database querying literacy and visual communication skills — students should leave able to write and interpret queries, and to design clear, honest data visuals.

## Modules

| Module | Topic | Assessment |
|--------|-------|------------|
| [01_Data](01_Data/) | Database concepts, SQL-style queries, joins, aggregation | Practical assessment |
| [02_Digital_Content](02_Digital_Content/) | Infographics, data visualisation, visual design principles | Practical assessment |

## Notes

Module 01 (Data) is query-heavy and directly prepares students for VBE Section 1 tasks. Module 02 (Digital Content) teaches visual honesty alongside design — pair the final integration task with a critique exercise to reinforce both.
