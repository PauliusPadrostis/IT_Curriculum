## Overview

Given messy + multi-source data: clean + combine + visualize + short conclusion.\
\

---

## 1️⃣ Completion Definition

Student independently performs a mini data workflow:

- select/receive dataset

- clean it appropriately

- justify source reliability

- provide a short conclusion grounded in data

## 2️⃣ Minimum Evidence

Submission includes:

- cleaned dataset

- cleaning log

- source quality justification

- 4–6 sentence conclusion with ≥1 numeric reference

## 3️⃣ Bloom's Taxonomy Level

**Bloom:** Apply → Analyze

## 4️⃣ Readiness Gate

✔ All true:

- cleaning is correct and documented

- conclusion includes numeric reference and stays within evidence

- reliability reasoning includes at least 1 limitation\
  If not → remediation required before Mini-block 2.
