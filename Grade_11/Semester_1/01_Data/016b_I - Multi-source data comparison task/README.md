# I — Multi-source data comparison task

**Type:** 🔗 Integration practice
**Grade:** Grade 11
**Module:** Data
**Semester:** Semester 1
**Status:** 🚧 WIP

---

## Overview

- Students receive two datasets, join them, and answer analytical questions
- Reinforces join/pivot skills with exam-style framing
- Practice combining data from multiple sources

## Sub-topics

- Data joining techniques
- Pivot table analysis on combined data
- Answering analytical questions from joined datasets

---

## 1️⃣ Completion Definition

Student can join two datasets and answer analytical questions about the combined data.

## 2️⃣ Minimum Evidence

Correctly joins datasets and provides accurate answers to at least 3 analytical questions.

## 3️⃣ Bloom's Taxonomy Level

Level 3 — Apply

## 4️⃣ Readiness Gate

Completed through lesson 016 (second practical assessment).

## 5️⃣ Common Failure Modes

- Incorrect join key selection
- Misinterpreting pivot table results
- Not validating that join produced expected row count
