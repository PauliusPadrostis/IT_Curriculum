# I — VBE1-style data interpretation drill

**Type:** 🔗 Integration practice
**Grade:** Grade 11
**Module:** Data
**Semester:** Semester 1
**Status:** 🚧 WIP

---

## Overview

- Students receive pre-computed tables and charts
- Answer "what does this show?" and "which query produced this?" questions
- Pure analytical reasoning, no spreadsheet manipulation
- Mirrors VBE Part 1 data interpretation question format

## Sub-topics

- Interpreting charts and tables
- Identifying queries from results
- Analytical reasoning about data presentations

---

## 1️⃣ Completion Definition

Student can interpret pre-computed data presentations and identify the queries or operations that produced them.

## 2️⃣ Minimum Evidence

Correctly answers at least 4 out of 5 data interpretation questions.

## 3️⃣ Bloom's Taxonomy Level

Level 3 — Apply

## 4️⃣ Readiness Gate

Completed through lesson 016 (second practical assessment).

## 5️⃣ Common Failure Modes

- Misreading chart axes or scales
- Confusing correlation with causation in data
- Not considering data filtering in query identification
