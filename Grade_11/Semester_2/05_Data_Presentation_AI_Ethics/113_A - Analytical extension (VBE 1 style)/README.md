## Overview

Structured questions: interpret outputs, pick correct steps, short answers + brief justification.\
\

---

## 1️⃣ Completion Definition

Student independently completes an assessment demonstrating:

- correct interpretation of a data presentation

- correct identification of a misleading element or risk

- ability to write a concise conclusion grounded in numbers

- ability to apply ethics/regulation reasoning to an AI/data scenario

## 2️⃣ Minimum Evidence

Assessment pack includes:

- ≥1 interpretation item (chart/table/map) requiring ≥1 numeric reference

- ≥1 “spot the misleading element / risk” item with justification

- ≥1 short argument prompt (ethics/regulation, AI/data viewpoint) requiring a counterpoint

Responses must be short-answer / exam style.

## 3️⃣ Bloom's Taxonomy Level

**Bloom:** Analyze → Evaluate

## 4️⃣ Readiness Gate

✔ All true:

- Numeric reference correct and relevant

- Correctly identifies at least one real misrepresentation risk

- Ethics/regulation response includes counterpoint\
  If not → targeted remediation required before end-of-year drill phase and Grade 12 data expectations.
