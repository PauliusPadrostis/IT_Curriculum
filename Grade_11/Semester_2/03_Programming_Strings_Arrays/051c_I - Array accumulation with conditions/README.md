# I — Array accumulation with conditions

**Type:** 🔗 Integration practice
**Grade:** Grade 11
**Module:** Programming – Strings & Arrays
**Semester:** Semester 2
**Status:** 🚧 WIP

---

## Overview

- Students traverse an array and apply conditional counting/summing
- Reinforces patterns from L block with hands-on practice
- Conditional accumulation: count elements meeting criteria, sum selected elements

## Sub-topics

- Array traversal with conditions
- Conditional counting and summing
- Combining multiple conditions

---

## 1️⃣ Completion Definition

Student can traverse an array, apply conditions, and compute conditional sums/counts.

## 2️⃣ Minimum Evidence

Working program that correctly computes conditional aggregates from an array.

## 3️⃣ Bloom's Taxonomy Level

Level 3 — Apply

## 4️⃣ Readiness Gate

Completed L block array lessons (up to lesson 051).

## 5️⃣ Common Failure Modes

- Incorrect loop bounds
- Logic errors in conditions (using AND vs OR)
- Not initializing accumulators
