# I — File-driven string and array task

**Type:** 🔗 Integration practice
**Grade:** Grade 11
**Module:** Programming – Strings & Arrays
**Semester:** Semester 2
**Status:** 🚧 WIP

---

## Overview

- Students read data from a file, build an array, compute aggregates
- Connects file I/O (Module 04) back to string/array skills
- Combines multiple skills: file reading, string parsing, array operations

## Sub-topics

- Reading structured data from files
- Building arrays from file data
- Computing aggregates on file-sourced arrays
- Integrating file I/O with string/array processing

---

## 1️⃣ Completion Definition

Student can read data from a file, store it in an array, and compute results using string/array operations.

## 2️⃣ Minimum Evidence

Working program that reads a file, builds an array, and produces correct aggregate output.

## 3️⃣ Bloom's Taxonomy Level

Level 3 — Apply

## 4️⃣ Readiness Gate

Completed first assessment block (through lesson 058) and Module 04 file I/O lessons.

## 5️⃣ Common Failure Modes

- Incorrect file reading causing array population errors
- Not handling end-of-file correctly
- Mixing up string parsing with array indexing
