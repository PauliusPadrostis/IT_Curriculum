# I — Code tracing mixed practice

**Type:** 🔗 Integration practice
**Grade:** Grade 11
**Module:** Programming – Strings & Arrays
**Semester:** Semester 2
**Status:** 🚧 WIP

---

## Overview

- Students trace 3–4 short programs involving strings and arrays
- Predict output, then verify by running
- Exam-critical skill: mental execution of code

## Sub-topics

- Tracing string operations step by step
- Tracing array operations step by step
- Predicting program output
- Identifying common tracing mistakes

---

## 1️⃣ Completion Definition

Student can accurately trace program execution and predict output for string and array programs.

## 2️⃣ Minimum Evidence

Correctly predicts output for at least 3 out of 4 tracing exercises before running the code.

## 3️⃣ Bloom's Taxonomy Level

Level 3 — Apply

## 4️⃣ Readiness Gate

Completed first assessment block (through lesson 058).

## 5️⃣ Common Failure Modes

- Losing track of variable values during iteration
- Confusing 0-based vs 1-based indexing
- Forgetting string immutability in some operations
