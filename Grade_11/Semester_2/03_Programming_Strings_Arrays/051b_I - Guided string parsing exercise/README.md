# I — Guided string parsing exercise

**Type:** 🔗 Integration practice
**Grade:** Grade 11
**Module:** Programming – Strings & Arrays
**Semester:** Semester 2
**Status:** 🚧 WIP

---

## Overview

- Students parse a structured string (e.g., "Name;Age;Grade") into components
- Apply validation to parsed data
- Bridges the L block to the I block in the string/array module

## Sub-topics

- String splitting by delimiter
- Data validation after parsing
- Structured string formats

---

## 1️⃣ Completion Definition

Student can parse a structured string into components and validate the results.

## 2️⃣ Minimum Evidence

Working program that correctly parses a semicolon-delimited string and validates extracted fields.

## 3️⃣ Bloom's Taxonomy Level

Level 3 — Apply

## 4️⃣ Readiness Gate

Completed L block string lessons (up to lesson 051).

## 5️⃣ Common Failure Modes

- Off-by-one errors in string position tracking
- Not handling missing delimiters
- Confusion between character position and substring extraction
