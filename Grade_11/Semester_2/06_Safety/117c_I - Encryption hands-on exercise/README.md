# I — Encryption hands-on exercise

**Type:** 🔗 Integration practice
**Grade:** Grade 11
**Module:** Safety – Advanced Topics
**Semester:** Semester 2
**Status:** 🚧 WIP

---

## Overview

- Students generate a key pair (using GPG4Win/Kleopatra or an online PGP tool)
- Exchange public keys with a partner
- Encrypt a short message for their partner, partner decrypts it
- Verify a website certificate in a browser and identify: issuer, expiry, domain match
- Short written reflection: "What would happen if someone intercepted the encrypted message without the private key?"

---

## 1️⃣ Completion Definition

Student has successfully encrypted and decrypted a message using public/private keys and can explain what they did.

## 2️⃣ Minimum Evidence

Screenshot or teacher verification of successful encrypt/decrypt exchange + correct certificate identification.

## 3️⃣ Bloom's Taxonomy Level

Level 3 — Apply

## 4️⃣ Readiness Gate

Completed the encryption concepts lesson.
