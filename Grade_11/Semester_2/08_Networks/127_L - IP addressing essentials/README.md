# L127 — IP addressing essentials

**Type:** 📖 Lesson
**Grade:** Grade 11
**Module:** Networks
**Semester:** Semester 2
**Sequence:** 127 of grade year
**Status:** 🚧 WIP

---

## Overview

IPv4 vs IPv6 conceptually; public vs private; static vs dynamic; local network reasoning.\
\

---

## 1️⃣ Completion Definition

Student can explain purpose of IP addresses, distinguish public vs private addressing, and determine whether devices are in the same network using basic subnet logic (conceptual level).

## 2️⃣ Minimum Evidence

Given 4 IP configuration cases, student identifies:\
Which addresses are private/public\
Whether two devices can communicate locally\
One likely misconfiguration cause if communication fails

## 3️⃣ Bloom's Taxonomy Level

Understand → Apply

## 4️⃣ Readiness Gate

Can the student determine same-network vs different-network cases without rote memorization errors? Yes/No
