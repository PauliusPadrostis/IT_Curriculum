# Grade 11 – Semester 2

**Semester goal:** Extend programming to collections and file I/O, broaden digital literacy (AI ethics, safety, networks), and finish with a full VBE diagnostic to identify gaps before Grade 12.

## Modules

| Module | Topic | Assessment |
|--------|-------|------------|
| [03_Programming_Strings_Arrays](03_Programming_Strings_Arrays/) | String operations, 1D arrays, traversal/aggregation | Practical assessment |
| [04_Programming_Insert_Delete](04_Programming_Insert_Delete/) | Array modification, file I/O pipeline | Practical assessment |
| [05_Data_Presentation_AI_Ethics](05_Data_Presentation_AI_Ethics/) | Presenting findings, AI-content ethics | Integration task |
| [06_Safety](06_Safety/) | Cybersecurity, data protection | Formative check |
| [07_Virtual_Communications](07_Virtual_Communications/) | Professional digital communication | Formative check |
| [08_Networks](08_Networks/) | Network types, protocols, IP basics | Formative check |
| [09_VBE_Diagnostic](09_VBE_Diagnostic/) | Full VBE-format diagnostic exam | Diagnostic report |

## Notes

Programming modules (03–04) should be completed before the diagnostic. Modules 05–08 can run in parallel with the tail of Module 04. The VBE Diagnostic (Module 09) is the key output of the semester — use results to plan Grade 12 remediation.
