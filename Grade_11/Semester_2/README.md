# Grade 11 – Semester 2

**Semester goal:** Extend programming to collections and file I/O, broaden digital literacy (AI ethics, safety, networks), and finish with a full VBE diagnostic to identify gaps before Grade 12.

## Modules

| Module | Topic | Delivery | Assessment |
|--------|-------|----------|------------|
| [03_Programming_Strings_Arrays](03_Programming_Strings_Arrays/) | String operations, 1D arrays, traversal/aggregation | In-class | Practical assessment |
| [04_Programming_Insert_Delete](04_Programming_Insert_Delete/) | Array modification, file I/O pipeline | In-class | Practical assessment |
| [05_Data_Presentation_AI_Ethics](05_Data_Presentation_AI_Ethics/) | Presenting findings, AI-content ethics | In-class | Integration task |
| [06_Safety](06_Safety/) | Cybersecurity, crypto, data protection | L = Theory Load; I/A = In-class | Mixed-format assessment |
| [07_Virtual_Communications](07_Virtual_Communications/) | Professional digital communication | L = Theory Load; I/A = In-class | Practical coordination check |
| [08_Networks](08_Networks/) | Network types, protocols, IP basics | L = Theory Load; I/A = In-class | Structured test |
| [09_VBE_Diagnostic](09_VBE_Diagnostic/) | Full VBE-format diagnostic exam | In-class | Diagnostic report |
| [10_Theory_Assessment](10_Theory_Assessment/) | Theory quizzes covering all theory-loaded content | In-class | Theory quizzes |

## Notes

Programming modules (03–04) should be completed before the diagnostic. Modules 05–08 can run in parallel with the tail of Module 04. The VBE Diagnostic (Module 09) is the key output of the semester — use results to plan Grade 12 remediation.

Modules 06, 07, and 08 use **Theory Load** delivery for their L-type lessons: students work through homework packages independently, and only I and A lessons are scheduled classroom sessions. Module 10 provides the in-class theory assessments that cover all theory-loaded content.
