# L9 (1) — Computer components overview

**Type:** 📖 Lesson
**Grade:** Grade 9 (I gimnazijos klasė)
**Module:** Technological Foundations
**Semester:** Semester 1
**Sequence:** 9 of grade year
**Lessons allocated:** 1
**Status:** 🚧 WIP

---

## Overview

CPU, RAM, storage, motherboard, input/output.

---

## 1️⃣ Completion Definition

Student identifies main internal components (CPU, RAM, storage, motherboard, PSU) and correctly explains their functional role in the data-processing cycle.

## 2️⃣ Minimum Evidence

Labeled system diagram + short functional explanation for each component (role, not brand/spec).

## 3️⃣ Bloom's Taxonomy Level

Understand → Explain.

## 4️⃣ Readiness Gate

If student confuses storage with RAM or cannot describe CPU function in simple terms, topic incomplete.
