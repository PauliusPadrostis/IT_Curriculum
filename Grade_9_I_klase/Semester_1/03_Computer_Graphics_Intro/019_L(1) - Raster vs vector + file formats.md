# L19 (1) — Raster vs vector + file formats

**Type:** 📖 Lesson
**Grade:** Grade 9 (I gimnazijos klasė)
**Module:** Computer Graphics Intro
**Semester:** Semester 1
**Sequence:** 19 of grade year
**Lessons allocated:** 1
**Status:** 🚧 WIP

---

## Context

JPG, PNG, SVG, PDF — when to use which.

## Completion Definition

Student distinguishes raster vs vector graphics by representation logic (pixels vs shapes) and selects appropriate common file formats for a stated purpose (web, print, editing, sharing).

## Minimum Evidence

Given 6 use-cases (e.g., logo, photo, screenshot, poster, icon set), student chooses raster/vector and an appropriate format, with one-sentence justification per case.

## Cognitive Level (Bloom's)

Understand → Apply

## Readiness Gate

Student can explain at least one quality consequence of wrong choice (e.g., pixelation, unnecessary file size, loss of editability).
