# A2 (1) — Practical data task (graded)

**Type:** 📝 Assessment
**Grade:** Grade 10 (II gimnazijos klasė)
**Module:** Data – Spreadsheet Foundations
**Semester:** Semester 1
**Status:** 🚧 WIP

---

## Context

Students must:

- Compute correctly with functions and references

- Use sort/filter to answer a question

- Provide either:

  - a chart with correct labeling **or**

  - a written conclusion grounded in computed results.

## Completion Definition

Student independently completes a structured data-analysis task integrating computation, sorting/filtering, and interpretation in a graded setting without guided prompts.

## Minimum Evidence

Given a dataset (30–50 rows), student:\
creates necessary computed columns using appropriate functions\
applies multi-criteria sorting or filtering aligned with a stated question\
produces correct numerical outputs\
writes a short structured interpretation tied directly to computed results\
maintains clean layout and readable structure.

## Cognitive Level (Bloom's)

Apply → Analyze (independent structured execution with reasoning).

## Readiness Gate

All required outputs are correct, formulas are structurally valid, and interpretation logically matches the produced data subset.
