---

## 1️⃣ Completion Definition

Student can analyse practical assessment.

## 2️⃣ Minimum Evidence

Given a medium-sized dataset (20–40 rows), student:\
separates inputs from computed outputs\
uses at least four different core functions appropriately\
avoids hardcoding values inside formulas\
produces correct numerical outputs\
maintains readable structure (clear labels, logical layout).

## 3️⃣ Bloom's Taxonomy Level

Apply (independent execution under assessment conditions).

## 4️⃣ Readiness Gate

All required outputs are correct and formulas reference appropriate ranges (no manual arithmetic substitutions).
