## Context

- One compact task including:

  - 1–2 functions

  - at least one absolute reference use

  - one filter or sort

  - one chart OR a short conclusion

- Teacher checklist: correctness + reference discipline + clarity.

## Completion Definition

Student completes a time-constrained spreadsheet task while explicitly applying a predefined quality checklist (structure, correct ranges, no hardcoding, logical sequencing).

## Minimum Evidence

Within a fixed time limit, student:\
builds formulas using correct ranges\
applies at least one sort or filter accurately\
checks for common structural errors (misaligned ranges, overwritten formulas, incorrect references)\
self-verifies results using the checklist before submission.

## Cognitive Level (Bloom's)

Apply (fluency under mild time pressure).

## Readiness Gate

Task is completed within time and passes all checklist items without structural errors (no broken references, no manual substitutions).
