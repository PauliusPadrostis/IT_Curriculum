# L8 (1) — Charts as communication (minimum viable)

**Semester:** Semester 1
**Sequence:** 8 of grade year

**Status:** 🚧 WIP

---

## Sub-topics

- Choosing chart type by question (bar/line/pie basics)

- Label discipline (titles, axes, units)

- Misleading charts (scale, missing labels).

## Completion Definition

Student selects and constructs an appropriate basic chart (column, line, pie) that communicates a specific analytical point rather than merely visualizing data mechanically.

## Minimum Evidence

Given computed data, student:\
chooses a chart type aligned with the question (comparison, trend, proportion)\
selects correct data ranges (no mislabeled axes)\
adds meaningful title and axis labels\
avoids decorative clutter that distorts interpretation\
explains why the chosen chart type fits the analytical goal.

## Cognitive Level (Bloom's)

Understand → Apply (visual reasoning aligned with analytical intent).

## Readiness Gate

Chart type and selected data must logically match the stated question (e.g., no pie chart for time trends, no trend line for unordered categories).
