# L1 (1) — Spreadsheet as a “computational model”

**Type:** 📖 Lesson
**Grade:** Grade 10 (II gimnazijos klasė)
**Module:** Data – Spreadsheet Foundations
**Semester:** Semester 1
**Sequence:** 1 of grade year
**Lessons allocated:** 1
**Status:** 🚧 WIP

---

## Sub-topics

- Cell as variable, formula as expression

- Relative vs absolute references (why they matter)

- Common errors (#DIV/0!, #VALUE!, wrong reference).

---

## 1️⃣ Completion Definition

Student explains and demonstrates that a spreadsheet is a rule-based computational system where cell references form dependency chains, and changing one input propagates deterministically through formulas.

## 2️⃣ Minimum Evidence

Student builds a small model (minimum 5–8 cells) where:\
inputs are clearly separated\
formulas reference other cells (not hardcoded values)\
a change in one input correctly updates all dependent outputs\
student can verbally trace one recalculation path.

## 3️⃣ Bloom's Taxonomy Level

Understand → Apply (computational dependency reasoning, not mechanical typing).

## 4️⃣ Readiness Gate

Student can correctly predict what will change (and what will not change) after modifying a specific input cell — without executing first.
