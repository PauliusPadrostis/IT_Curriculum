## Completion Definition

Student correctly uses core spreadsheet functions (SUM, AVERAGE, MIN, MAX, COUNT, basic IF) to extract structured information from a dataset and selects functions based on analytical intent rather than imitation.

## Minimum Evidence

Given a 15–30 row dataset, student:\
writes at least five different function-based formulas\
uses proper cell ranges (not manual addition)\
uses at least one logical condition (IF)\
explains why a chosen function matches the question being asked.

## Cognitive Level (Bloom's)

Apply → Analyze (function selection aligned with question logic).

## Readiness Gate

When given a natural-language analytical question, student can independently select the correct function type before writing the formula.
