# L3 (1) — Sorting + filtering as question logic

**Type:** 📖 Lesson
**Grade:** Grade 10 (II gimnazijos klasė)
**Module:** Data – Spreadsheet Foundations
**Semester:** Semester 1
**Status:** 🚧 WIP

---

## Sub-topics

- Sort by one/two keys

- Filter as “selection criteria”

- Translating a prompt into filter rules (AND/OR conceptually).

## Completion Definition

Student understands sorting and filtering not as mechanical interface actions but as question-formulation tools used to isolate patterns, extremes, subsets, and comparisons within a dataset.

## Minimum Evidence

Given a structured dataset, student:\
applies single- and multi-criteria sorting correctly\
uses filtering to isolate records matching explicit conditions\
explains what question each sort/filter operation answers\
identifies when sorting changes order but not values.

## Cognitive Level (Bloom's)

Understand → Analyze (question-driven data manipulation).

## Readiness Gate

Student can translate a natural-language question (e.g., “Which category has the highest average under condition X?”) into an appropriate sort or filter strategy without trial-and-error clicking.
