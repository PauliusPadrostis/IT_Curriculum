## 1️⃣ Completion Definition

Student completes short integrated programming tasks (sequence + variables + conditionals) within a fixed time limit while maintaining logical correctness and clean structure.

## 2️⃣ Minimum Evidence

Within the time constraint, student:\
writes a complete program solving a clearly defined task\
correctly handles input, processing, and output\
implements at least one conditional branch correctly\
avoids syntax errors and unresolved runtime errors\
produces correct output for at least two distinct test inputs.

## 3️⃣ Bloom's Taxonomy Level

Apply (fluency under time pressure).

## 4️⃣ Readiness Gate

Program compiles/runs without errors and produces correct outputs for all provided test cases within the allotted time.
