## Context

Students receive:

- Broken programs

- Off-by-one errors

- Wrong condition logic

They must:

- Identify bug type

- Fix and explain.\
  \

## Completion Definition

Student systematically identifies and corrects syntax, logical, and boundary errors in short programs, demonstrating structured debugging rather than random trial-and-error edits.

## Minimum Evidence

Student:\
  locates at least one syntax error and corrects it\
  identifies at least one logical error (incorrect condition or formula)\
  tests program with multiple inputs\
  explains why the original version failed\
  produces corrected version with verified outputs.

## Cognitive Level (Bloom's)

Analyze → Evaluate (error detection and correction reasoning).

## Readiness Gate

Student can explain the root cause of the error and demonstrate that the corrected program produces expected results for all provided test cases.
