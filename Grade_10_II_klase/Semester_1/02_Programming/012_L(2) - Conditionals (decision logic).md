**Type:** 📖 Lesson
**Grade:** Grade 10 (II gimnazijos klasė)
**Module:** Programming – Zero-to-Code & Conditionals
**Semester:** Semester 1
**Sequence:** 12 of grade year
**Lessons allocated:** 2
**Status:** 🚧 WIP

---

## Sub-topics

- if / else structure

- Comparison operators

- Nested conditions (light exposure)

- Common beginner errors (== vs =, indentation, braces).

## Completion Definition

Student correctly implements decision structures (if / else / elif) to control program flow based on logical conditions and understands boolean evaluation as the mechanism behind branching.

## Minimum Evidence

Student:\
writes programs using single and multi-branch conditionals\
constructs correct logical expressions (comparisons, AND/OR where applicable)\
avoids mutually exclusive logic errors\
traces branching paths correctly for given inputs\
explains which condition is evaluated and why a branch executes.

## Cognitive Level (Bloom's)

Apply → Analyze (branch reasoning, not syntax copying).

## Readiness Gate

Given a conditional program and specific input values, student can correctly predict which branch executes and what output is produced.
