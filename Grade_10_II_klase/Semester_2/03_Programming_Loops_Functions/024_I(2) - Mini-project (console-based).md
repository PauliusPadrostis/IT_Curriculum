## Completion Definition

Student designs and implements a small console-based program that integrates variables, conditionals, loops, and user-defined functions into a coherent, specification-driven solution.

## Minimum Evidence

Student:\
interprets a multi-part specification correctly\
structures the program using modular decomposition\
uses loops and conditionals appropriately\
ensures consistent input/output formatting\
tests program with multiple input scenarios\
corrects detected logical or boundary errors\
produces fully correct output according to requirements.

## Cognitive Level (Bloom's)

Apply → Analyze (integrated algorithm construction and structural reasoning).

## Readiness Gate

Program runs without errors, satisfies all specification requirements, and passes defined test scenarios without structural flaws.
