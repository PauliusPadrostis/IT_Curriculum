## Completion Definition

Student independently solves a programming task requiring correct use of loops for aggregation, counting, or simple repetitive logic under assessment conditions.

## Minimum Evidence

Student:\
implements at least one loop structure correctly\
uses accumulator variables appropriately\
avoids boundary errors (off-by-one, incorrect start/end)\
produces correct output for multiple test cases\
maintains clean and readable structure.

## Cognitive Level (Bloom's)

Apply → Analyze (independent loop-based problem solving).

## Readiness Gate

Program executes without errors and produces correct results for all specified test inputs, including edge cases (e.g., minimum value range).
