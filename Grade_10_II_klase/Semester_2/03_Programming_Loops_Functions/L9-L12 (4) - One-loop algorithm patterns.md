# L9–L12 (4) — One-loop algorithm patterns

**Type:** 📖 Lesson
**Grade:** Grade 10 (II gimnazijos klasė)
**Module:** Programming – Loops, Algorithms & Functions
**Semester:** Semester 2
**Status:** 🚧 WIP

---

## Sub-topics

- Sum / count

- Minimum / maximum

- Average (derived value logic)

- Linear search (find element / flag variable)

- “Count if…” pattern

These are explicitly required Grade 10 algorithm types

Students must master patterns, not memorize tasks.\
\

## Completion Definition

Student recognizes and correctly implements core single-loop algorithm patterns: sum, count, average, minimum/maximum tracking, conditional counting, and simple linear search.

## Minimum Evidence

Student:\
implements at least four distinct loop-based patterns\
initializes control and tracking variables correctly\
updates tracking variables inside the loop appropriately\
handles empty or boundary conditions logically\
can explain which pattern fits which problem type.

## Cognitive Level (Bloom's)

Apply → Analyze (pattern recognition and correct selection).

## Readiness Gate

Given a problem description, student can select the correct single-loop pattern before writing code and produce correct output for all test cases.
