# I1–I2 (2) — Combined loop problems

**Type:** 🔗 Integration
**Grade:** Grade 10 (II gimnazijos klasė)
**Module:** Programming – Loops, Algorithms & Functions
**Semester:** Semester 2
**Status:** 🚧 WIP

---

## Context

- Multi-pattern tasks (e.g., count + min + search in one problem)
- Students must plan before coding
- Emphasis: structured thinking.

## Completion Definition

Student integrates multiple one-loop patterns within a single program (e.g., sum + conditional count, min + average, search + aggregate) while preserving logical correctness and variable discipline.

## Minimum Evidence

Student:\
implements at least two tracking patterns in one loop\
correctly initializes all tracking variables\
avoids overwriting values unintentionally\
maintains correct update logic for each tracked metric\
produces correct output for multiple test cases\
can explain how each tracked value is updated during iteration.

## Cognitive Level (Bloom's)

Analyze → Apply (compositional loop reasoning).

## Readiness Gate

Program produces correct results for all required outputs simultaneously, and student can trace one full iteration cycle explaining every variable update.
