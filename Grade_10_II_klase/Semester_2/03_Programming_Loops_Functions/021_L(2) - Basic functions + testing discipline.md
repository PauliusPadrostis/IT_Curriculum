## Completion Definition

Student correctly defines and calls simple functions that return a value, separates computation from main program flow, and systematically tests function behavior using multiple input cases.

## Minimum Evidence

Student:\
writes at least one function that returns a value\
passes arguments correctly\
uses return values in the main program\
avoids global-variable dependence for core logic\
tests function with at least three distinct inputs\
identifies and corrects at least one logic error through testing.

## Cognitive Level (Bloom's)

Apply → Analyze (modular reasoning and verification discipline).

## Readiness Gate

Function produces correct results for all provided test cases, and student can explain the purpose of parameters and return value clearly.
