## Completion Definition

Student correctly implements `for` and `while` loops to perform repeated computation, understands iteration control (initialization, condition, update), and avoids infinite loops and off-by-one errors.

## Minimum Evidence

Student:\
writes loops that iterate over numeric ranges correctly\
uses accumulator variables (sum, count, product)\
predicts number of iterations before execution\
avoids infinite loops\
correctly handles inclusive/exclusive boundaries\
traces loop state changes step by step.

## Cognitive Level (Bloom's)

Apply → Analyze (mechanical control of repetition).

## Readiness Gate

Given a loop snippet, student can accurately state how many iterations execute and what the final accumulator value will be.
