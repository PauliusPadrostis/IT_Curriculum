# Grade 12 – Advanced Data · Advanced Programming · VBE Exam Prep

**Year focus:** Exam-grade mastery — multi-step data reporting, file-based programming with sorting and searching algorithms, and sustained VBE mock practice under timed conditions.

## Modules

| Semester | Module | Topic |
|----------|--------|-------|
| 1 | [01_Data_Reporting](Semester_1/01_Data_Reporting/) | Multi-criteria filtering, pivot reports, chart selection, switching techniques |
| 1 | [02_Programming](Semester_1/02_Programming/) | File I/O, tokenization, multi-function programs, formatted output |
| 2 | [03_Programming_Advanced](Semester_2/03_Programming_Advanced/) | Sorting algorithms, collection operations, completeness and edge-case thinking |
| 2 | [04_Exam_Prep](Semester_2/04_Exam_Prep/) | Timed tasks, mocks, error clinics, station-rotation drills |

## Progression logic

Semester 1 consolidates both data and programming at exam level; every task mirrors VBE task structure and marking criteria. Semester 2 adds sorting/searching, then shifts entirely to exam simulation — mocks, targeted error clinics, and pacing drills.

## Assessment points

- Practical assessments in every module
- Timed tasks in Module 04 (VBE format)
- Two full mock sessions (lessons 119–120)
- Error clinics and diagnostic rotation drills (lessons 122–125)
