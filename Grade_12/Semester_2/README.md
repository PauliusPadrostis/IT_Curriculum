## Notes

Module 05 contains compressed lessons (066, 085, 086) that merge previously separate topics. It should be fully complete before the first mock (lesson 113). Error clinics (lessons 122–125) are targeted at the specific mistake patterns identified in mocks — run them after each mock, not before.

Module 06 Exam Prep covers both practical exam skills and theory exam preparation — T-type timed lessons include theory questions from Theory Load packs alongside data and programming tasks.
