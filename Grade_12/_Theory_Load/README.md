# Grade 12 — Theory Load Packs

Theory Load packs deliver curriculum-required theory content as structured homework. These topics appear in VBE exams as theory questions (VBE Part 2 Section I: 8–12 marks of short/open answer). Theory drilling is integrated into Module 06 (Exam Prep) D/T/G lessons.
