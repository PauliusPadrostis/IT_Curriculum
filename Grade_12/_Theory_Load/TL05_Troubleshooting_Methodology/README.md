# TL05 — Troubleshooting Methodology

**NŠA requirement:** Skaitmeninių įrenginių sutrikimų aptarimas
**Delivery:** Homework reading + self-check quiz

## Key Concepts Students Must Know

- Systematic troubleshooting process (identify symptoms, hypothesize, test, resolve)
- Using forums and community resources for problem solving
- Effective search query formulation for technical problems
- Translation tools for accessing non-Lithuanian resources
- When to escalate vs self-resolve

## Exam Relevance

Short answer question in VBE Part 2 Section I on troubleshooting approach or resource identification.
