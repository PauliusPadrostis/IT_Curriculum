# TL03 — Animation Concepts

**NŠA requirement:** Animuotų objektų kūrimas
**Delivery:** Homework reading + self-check quiz
**Estimated reading time:** 15 min

---

## Key Concepts Students Must Know

- Frame-based animation: frames per second, frame sequences
- Keyframes and tweening (in-between generation)
- Motion, timing, lip sync, deformation (conceptual level)
- Animation software: Blender, Synfig, After Effects (awareness)
- Animation file formats and their uses

## Exam Relevance

1 question in VBE Part 1 on animation concepts or software identification.
