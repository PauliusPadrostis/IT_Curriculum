# I — Create a short digital publication

**Type:** 🔗 Integration practice
**Grade:** Grade 12
**Module:** E-Publishing Refresher
**Semester:** Semester 1
**Status:** 🚧 WIP
**Sequence:** 35 of grade year

---

## Overview

- Students create a 2–4 page digital publication using Canva or BookCreator
- Incorporating text, images, and at least one embedded media element
- Focus on layout, visual hierarchy, and export quality
- Connects to Grade 9 skills

## Sub-topics

- Digital publication creation workflow
- Layout and visual hierarchy
- Media embedding
- Export and quality considerations

---

## 1️⃣ Completion Definition

Student creates a multi-page digital publication with text, images, and embedded media that demonstrates good layout principles.

## 2️⃣ Minimum Evidence

Exported publication (PDF or link) with 2–4 pages, proper layout, and at least one embedded media element.

## 3️⃣ Bloom's Taxonomy Level

Level 3 — Apply

## 4️⃣ Readiness Gate

Completed e-publication concepts lesson.

## 5️⃣ Common Failure Modes

- Poor visual hierarchy (everything same size/weight)
- Not considering export format quality
- Media elements that don't enhance the content
