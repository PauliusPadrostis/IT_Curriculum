# L — Animation concepts for digital media

**Type:** 📖 Lesson
**Grade:** Grade 12
**Module:** E-Publishing Refresher
**Semester:** Semester 1
**Status:** 🚧 WIP
**Sequence:** 34 of grade year

---

## Overview

- What animation is (frame sequences, keyframes, tweening)
- Animation software landscape (Blender, Synfig, After Effects — awareness, not mastery)
- Elements of animation (motion, timing, lip sync, deformation — conceptual)
- Where animation is used in digital publications and media
- Animation file formats

## Sub-topics

- Frame-based animation principles
- Keyframes and tweening
- Animation software overview
- Animation in digital media context
- File formats for animation

---

## 1️⃣ Completion Definition

Student can explain keyframe animation, name animation software tools, and describe where animation is used in digital media.

## 2️⃣ Minimum Evidence

Correctly explains the difference between keyframe and tweened animation and names at least 2 animation tools.

## 3️⃣ Bloom's Taxonomy Level

Level 2 — Understand

## 4️⃣ Readiness Gate

None (standalone theory topic).

## 5️⃣ Common Failure Modes

- Confusing animation principles with video editing
- Thinking animation requires programming
- Not understanding the conceptual level expected (awareness, not mastery)
