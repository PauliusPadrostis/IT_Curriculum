# Grade 12 – Semester 1

**Semester goal:** Reach exam-level fluency in both data reporting and file-based programming — every task should mirror VBE task structure and marking criteria.

## Modules

| Module | Topic | Assessment |
|--------|-------|------------|
| [01_Data_Reporting](01_Data_Reporting/) | Multi-criteria filtering, pivot reports, chart selection | Practical assessments |
| [02_Programming](02_Programming/) | File I/O, tokenization, multi-function programs, formatted output | Practical assessments + timed drills |

## Notes

Both modules run at VBE pace from the start. Timed drills in Module 02 (lessons 044–045) build speed and accuracy under pressure. Use the integration tasks to simulate cross-section exam conditions before Semester 2.
