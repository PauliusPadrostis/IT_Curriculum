# Grade 12 – Semester 1

**Semester goal:** Reach exam-level fluency in both data reporting and file-based programming — every task should mirror VBE task structure and marking criteria.

## Modules

| Module | Topic | Assessment |
|--------|-------|------------|
| [01_Data_Reporting](01_Data_Reporting/) | Multi-criteria filtering, pivot reports, chart selection | Practical assessments |
| [02_AI_Practical](02_AI_Practical/) | Classification and prediction with Orange Data Mining (GUI-based ML) | Practical assessments |
| [03_E_Publishing_Refresher](03_E_Publishing_Refresher/) | E-publication concepts, animation concepts, practical digital publication task | Practical assessments |
| [04_Programming](04_Programming/) | File I/O, tokenization, multi-function programs, formatted output | Practical assessments + timed drills |

## Notes

All modules run at VBE pace from the start. Timed drills in Module 04 (lessons 045–046) build speed and accuracy under pressure. Use the integration tasks to simulate cross-section exam conditions before Semester 2.

Module 04 Programming contains compressed lessons (036, 037, 038, 039) that merge previously separate topics for tighter coverage.
