## 1️⃣ Completion Definition

Student can explain subtotal logic: counts/sums/averages by group.

## 2️⃣ Minimum Evidence

Correctly explains subtotal logic: counts/sums/averages by group in own words or in response to targeted questions without reference.

## 3️⃣ Bloom's Taxonomy Level

Level 2 — Understand / Remember

## 4️⃣ Readiness Gate

✔ Report meets all structural conditions:

- Correct grouping hierarchy

- Correct subtotal function

- No double-counting

- Grand total matches sum of subtotals

- Groups correctly collapsible

Any mismatch → Gate not met.

---

## 5️⃣ Common Failure Modes

- Grouping without sorting first

- Wrong aggregation function (e.g., count instead of sum)

- Misinterpreting category hierarchy

- Breaking dataset structure after grouping

- Not validating subtotal vs grand total consistency

- Manual calculation instead of structural grouping
