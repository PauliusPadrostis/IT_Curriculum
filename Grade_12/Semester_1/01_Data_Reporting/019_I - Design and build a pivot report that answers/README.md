## 1️⃣ Completion Definition

Student can apply design and build a pivot report that answers a clearly stated question.

## 2️⃣ Minimum Evidence

Independently completes a task involving design and build a pivot report that answers a clearly stated question without reference materials.

## 3️⃣ Bloom's Taxonomy Level

Level 3 — Apply

## 4️⃣ Readiness Gate

✔ All must be true:

- Dataset is pivot-ready (single header row, consistent types, no structural breaks)

- Pivot matches specification (field placement + aggregation)

- Chart type fits question (comparison/trend/composition as required)

- Conclusion includes ≥1 correct numeric reference and no unsupported claims

- Refresh validation succeeds (pivot/chart update confirmed)

Any failure → Gate not met.

---

## 5️⃣ Common Failure Modes

- Pivot built on dirty structure → wrong counts/aggregations

- Misplacing fields (e.g., metric in Rows instead of Values)

- Wrong aggregation default (count vs sum)

- Chart created from raw data instead of pivot results

- Conclusion overclaims causation or ignores filtering context

- Forgetting refresh → static report mistaken as dynamic
