# A28 — Practical Data Task (Graded)

**Type:** 📝 Assessment
**Grade:** Grade 12
**Module:** Data – Reporting & Analysis
**Semester:** Semester 1
**Sequence:** 28 of grade year
**Status:** 🚧 WIP

---

## Overview

Must include: multi-criteria selection, grouping/subtotals or pivot, one visualization, one conclusion paragraph.

---

## 1️⃣ Completion Definition

Student independently completes a full practical data task under assessment conditions, demonstrating:

- Structurally correct dataset preparation

- Correct method selection (filter/sort, subtotals, or pivot as required by task)

- Accurate aggregation and reporting mechanics

- Appropriate chart selection (if required)

- Concise, numerically grounded written conclusions

Performance must show integrated competence across the entire Data block.

---

## 2️⃣ Minimum Evidence

Student submits:

- Final cleaned dataset

- Required report output(s) (subtotals or pivot as specified)

- Chart (if required by task)

- Written conclusion(s) including ≥1 numeric reference per analytical question

Assessment conditions:

- No scaffolding

- No checklist access

- Independent work

---

## 3️⃣ Bloom's Taxonomy Level

**Bloom: Apply → Analyze → Evaluate**

- Apply correct tool mechanics

- Analyze reporting requirements

- Evaluate findings and justify conclusions numerically

Aligned to VBE 2 data practical expectations.

---

## 4️⃣ Readiness Gate

✔ All must be true:

- Dataset structurally valid

- Correct method chosen per task requirement

- Aggregations mathematically correct

- Chart technically correct (if required)

- Conclusions numerically supported

- No logical contradictions

If any structural or logical error present → Performance below mastery threshold.

---

## 5️⃣ Common Failure Modes

- Wrong tool selection (subtotals instead of pivot or vice versa)

- Aggregation mismatch (count vs sum)

- Chart mismatch with task intent

- Unsupported claims in conclusions

- Structural cleaning omitted under time pressure

- Numeric inconsistency between report and written answer
