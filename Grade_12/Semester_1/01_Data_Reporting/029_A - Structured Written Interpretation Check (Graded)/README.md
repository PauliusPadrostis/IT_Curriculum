# A29 — Structured Written / Interpretation Check (Graded)

**Type:** 📝 Assessment
**Grade:** Grade 12
**Module:** Data – Reporting & Analysis
**Semester:** Semester 1
**Sequence:** 29 of grade year
**Status:** 🚧 WIP

---

## Overview

- Short/structured questions: interpret a pivot/table/chart, identify the correct operation, justify a conclusion. Use charts / data tables produced in A1.

*(This mirrors the mixed nature of VBE 2: practical data work + reasoning/answers.)*

---

## 1️⃣ Completion Definition

Student independently produces accurate written interpretations of given data outputs (tables/pivot extracts/charts) by:

- Correctly extracting key values and relationships from the provided visual/table

- Answering structured questions with precise, data-grounded statements

- Providing at least one justified conclusion that matches the evidence

- Identifying at least one limitation, anomaly, or data-quality caveat when applicable

No tool execution required — interpretation accuracy and reasoning clarity are assessed.

---

## 2️⃣ Minimum Evidence

Student submits a written response set that includes:

- Direct numeric answers where required (values, differences, percentages)

- 2–4 sentence explanations for “why/how” prompts

- At least one conclusion that includes:

  - ≥1 numeric reference

  - an explicit link to the provided chart/table feature (trend, comparison, outlier, subgroup)

No sentence starters. No guided scaffolds.

---

## 3️⃣ Bloom's Taxonomy Level

**Bloom: Analyze → Evaluate**

- Analyze given representations (table/pivot/chart)

- Evaluate what claims are justified

- Communicate conclusions precisely

Aligned with VBE-style structured/atvirojo atsakymo expectations.

---

## 4️⃣ Readiness Gate

✔ All must be true:

- All numeric extractions are correct

- Explanations match the data shown (no invented causes)

- Conclusion contains ≥1 correct numeric reference

- Any limitation/caveat stated is logically valid (not generic filler)

Any incorrect extraction or unsupported inference → Gate not met.

---

## 5️⃣ Common Failure Modes

- Misreading axes / scale / units

- Confusing absolute values with percentages (or vice versa)

- Stating causation instead of correlation

- Writing generic conclusions without numeric evidence

- Ignoring subgroup differences (overgeneralizing)

- Missing or incorrect interpretation of outliers / missing data
