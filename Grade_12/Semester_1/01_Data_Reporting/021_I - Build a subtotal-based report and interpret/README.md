## 1️⃣ Completion Definition

Student can apply build a subtotal-based report and interpret the results.

## 2️⃣ Minimum Evidence

Independently completes a task involving build a subtotal-based report and interpret the results without reference materials.

## 3️⃣ Bloom's Taxonomy Level

Level 3 — Apply

## 4️⃣ Readiness Gate

✔ All must be true:

- Both reports are technically correct and internally consistent

- Differences identified are structurally valid (not superficial formatting)

- Chosen method aligns logically with scenario requirement

- Justification contains explicit reference to method capability (e.g., refreshability, dynamic dimensions, grouping rigidity)

If comparison is preference-based or structurally incorrect → Gate not met.

---

## 5️⃣ Common Failure Modes

- Comparing formatting instead of mechanics

- Claiming pivot is “better” without specifying why

- Not recognizing that subtotals require pre-sorting

- Failing to understand pivot dimensional flexibility

- Producing only one correct report (comparison invalid)

- Writing generic evaluation without scenario grounding
