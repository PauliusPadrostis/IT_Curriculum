## 1️⃣ Completion Definition

Student can explain writing conclusions grounded in computed evidence (not vibes).

## 2️⃣ Minimum Evidence

Correctly explains writing conclusions grounded in computed evidence (not vibes) in own words or in response to targeted questions without reference.

## 3️⃣ Bloom's Taxonomy Level

Level 2 — Understand / Remember

## 4️⃣ Readiness Gate

✔ All must be true:

- Chart type matches task intent (trend / comparison / composition / distribution)

- Chart has correct labels + units + readable scale

- Data range is correct (no missing or extra rows)

- Written conclusion contains:

  - ≥1 numeric reference

  - no unsupported inference

If any condition fails → Gate not met.

---

## 5️⃣ Common Failure Modes

- Choosing a visually attractive chart rather than correct chart type

- Misleading scale (truncated axis without justification)

- Missing labels/units making interpretation impossible

- Wrong data range (headers included as data, partial selection)

- Conclusion restates chart title but gives no quantified finding

- Overclaiming causation from correlation

- Ignoring outliers or missing data effects
