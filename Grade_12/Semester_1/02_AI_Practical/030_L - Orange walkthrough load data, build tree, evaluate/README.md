# L — Orange walkthrough: load data, build tree, evaluate

**Type:** 📖 Lesson
**Grade:** Grade 12
**Module:** AI Practical — Classification and Prediction
**Semester:** Semester 1
**Status:** 🚧 WIP
**Sequence:** 30 of grade year

---

## Overview

- Open Orange Data Mining
- Load a sample dataset
- Connect widgets: File → Data Table → Tree → Test & Score
- Read the accuracy output
- Visualize the decision tree

## Sub-topics

- Orange Data Mining interface
- Widget connection workflow
- Data Table widget
- Tree classifier widget
- Test & Score evaluation widget

---

## 1️⃣ Completion Definition

Student can follow along to build a classification pipeline in Orange and read the accuracy result.

## 2️⃣ Minimum Evidence

Successfully connects widgets and produces an accuracy score for a sample dataset.

## 3️⃣ Bloom's Taxonomy Level

Level 2–3 — Understand / Apply

## 4️⃣ Readiness Gate

Completed lesson on ML classification concepts.

## 5️⃣ Common Failure Modes

- Connecting widgets in wrong order
- Not understanding what the accuracy number means
- Confusion about widget inputs vs outputs
