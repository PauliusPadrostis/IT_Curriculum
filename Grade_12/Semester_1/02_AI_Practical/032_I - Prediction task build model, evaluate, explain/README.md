# I — Prediction task: build model, evaluate, explain

**Type:** 🔗 Integration practice
**Grade:** Grade 12
**Module:** AI Practical — Classification and Prediction
**Semester:** Semester 1
**Status:** 🚧 WIP
**Sequence:** 32 of grade year

---

## Overview

- Students receive a new dataset (e.g., weather data for temperature prediction, or student performance data)
- Build a model, evaluate it
- Compare two different algorithms (tree vs kNN)
- Write a short explanation of which performed better and why

## Sub-topics

- Working with unfamiliar datasets
- Algorithm comparison (tree vs kNN)
- Model evaluation and comparison
- Written analysis of results

---

## 1️⃣ Completion Definition

Student builds two models on a new dataset, compares their accuracy, and explains which performs better.

## 2️⃣ Minimum Evidence

Two models built and evaluated + written comparison explaining which is better and why.

## 3️⃣ Bloom's Taxonomy Level

Level 3–4 — Apply / Analyze

## 4️⃣ Readiness Gate

Completed Iris classification exercise.

## 5️⃣ Common Failure Modes

- Not knowing how to load a non-standard dataset
- Superficial comparison (just citing numbers without reasoning)
- Not considering why one algorithm might suit the data better
