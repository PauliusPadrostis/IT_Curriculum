# I — Classify Iris dataset: build, test, interpret accuracy

**Type:** 🔗 Integration practice
**Grade:** Grade 12
**Module:** AI Practical — Classification and Prediction
**Semester:** Semester 1
**Status:** 🚧 WIP
**Sequence:** 31 of grade year

---

## Overview

- Students independently load the Iris dataset in Orange
- Build a classification model
- Split into train/test
- Evaluate accuracy
- Write 2–3 sentences explaining what the model does and how well it performs

## Sub-topics

- Independent Orange workflow construction
- Iris dataset exploration
- Model evaluation interpretation
- Written explanation of results

---

## 1️⃣ Completion Definition

Student independently builds a classification model for Iris data and can explain its accuracy in writing.

## 2️⃣ Minimum Evidence

Working Orange pipeline with accuracy output + written explanation (2–3 sentences).

## 3️⃣ Bloom's Taxonomy Level

Level 3 — Apply

## 4️⃣ Readiness Gate

Completed Orange walkthrough lesson.

## 5️⃣ Common Failure Modes

- Copying walkthrough without understanding
- Unable to explain accuracy in own words
- Not splitting data properly
