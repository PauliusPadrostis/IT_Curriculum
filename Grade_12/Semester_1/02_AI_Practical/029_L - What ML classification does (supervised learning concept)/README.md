# L — What ML classification does (supervised learning concept)

**Type:** 📖 Lesson
**Grade:** Grade 12
**Module:** AI Practical — Classification and Prediction
**Semester:** Semester 1
**Status:** 🚧 WIP
**Sequence:** 29 of grade year

---

## Overview

- What is classification (sorting data into categories)
- What is a training set vs test set
- What "accuracy" means
- Decision tree as a visual model
- Brief mention of neural networks

## Sub-topics

- Supervised learning concept
- Training set vs test set
- Accuracy as a metric
- Decision tree visualization
- Neural networks (awareness level)

---

## 1️⃣ Completion Definition

Student can explain what classification is, distinguish training from test data, and describe what accuracy measures.

## 2️⃣ Minimum Evidence

Correctly answers: "What is the purpose of a test set?" and "What does 85% accuracy mean?"

## 3️⃣ Bloom's Taxonomy Level

Level 2 — Understand

## 4️⃣ Readiness Gate

None (standalone conceptual introduction).

## 5️⃣ Common Failure Modes

- Confusing training and test sets
- Thinking higher accuracy always means better model
- Not understanding that classification requires labeled data
