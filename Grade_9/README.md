# Grade 9 – Safety · Hardware · Graphics · Specialisation

**Year focus:** Foundational digital literacy — safe and responsible computer use, understanding hardware, and first creative applications (graphics, print, web, project).

## Modules

| Semester | Module | Topic |
|----------|--------|-------|
| 1 | [01_Safety](Semester_1/01_Safety/) | Ergonomics, privacy, account safety |
| 1 | [02_Technological_Foundations](Semester_1/02_Technological_Foundations/) | Hardware, software, file systems, OS |
| 1 | [03_Computer_Graphics_Intro](Semester_1/03_Computer_Graphics_Intro/) | Raster vs vector, basic image editing |
| 2 | [04_Computer_Graphics_Practical](Semester_2/04_Computer_Graphics_Practical/) | Applied graphics tasks and composition |
| 2 | [05_Specialisation_Digital_Printing](Semester_2/05_Specialisation_Digital_Printing/) | Print layout, colour, file preparation |
| 2 | [06_Specialisation_Website_Design](Semester_2/06_Specialisation_Website_Design/) | HTML/CSS basics, page structure |
| 2 | [07_Project](Semester_2/07_Project/) | Integrated project: plan, create, present |

## Progression logic

Students arrive with general computer familiarity but no formal IT study. Semester 1 builds safe habits and conceptual foundations; Semester 2 applies them in three parallel specialisation tracks before converging in a project.

## Assessment points

- Formative checks within each module
- Practical assessments at the end of Graphics and each Specialisation module
- Integrative project in Module 07
