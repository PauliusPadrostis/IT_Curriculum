# Grade 9 – Semester 1

**Semester goal:** Establish safe digital habits and a working mental model of how computers work, culminating in first practical graphics skills.

## Modules

| Module | Topic | Assessment |
|--------|-------|------------|
| [01_Safety](01_Safety/) | Ergonomics, privacy, account safety | Formative check |
| [02_Technological_Foundations](02_Technological_Foundations/) | Hardware, software, file systems, OS | Formative check |
| [03_Computer_Graphics_Intro](03_Computer_Graphics_Intro/) | Raster vs vector, basic image editing | Practical assessment |

## Notes

Safety and Technological Foundations pair well and can be co-scheduled for their assessments. Graphics builds directly on file system knowledge from Module 02.
