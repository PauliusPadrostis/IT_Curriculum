# L20 — Resolution & color basics

**Type:** 📖 Lesson
**Grade:** Grade 9
**Module:** Computer Graphics Intro
**Semester:** Semester 1
**Sequence:** 20 of grade year
**Status:** 🚧 WIP

---

## Overview

- Screen vs print logic
- RGB vs CMYK awareness.

---

## 1️⃣ Completion Definition

Student explains how resolution (pixel dimensions, PPI/DPI as context) and color choices affect clarity and usability, and can apply basic adjustments appropriately for simple tasks.

## 2️⃣ Minimum Evidence

Student completes a mini-task: resize/crop an image for a given target use (screen vs print) and explains the trade-off (quality vs file size, readability).

## 3️⃣ Bloom's Taxonomy Level

Apply → Analyze

## 4️⃣ Readiness Gate

Student can predict whether an image will look blurred/pixelated after a given scaling action and justify why.
