# Grade 9 – Semester 2

**Semester goal:** Apply graphics and digital skills in three specialisation tracks (print, web, open project), then integrate everything in a final project.

## Modules

| Module | Topic | Assessment |
|--------|-------|------------|
| [04_Computer_Graphics_Practical](04_Computer_Graphics_Practical/) | Applied graphics, composition, export | Practical assessment |
| [05_Specialisation_Digital_Printing](05_Specialisation_Digital_Printing/) | Print layout, colour modes, file prep | Practical assessment |
| [06_Specialisation_Website_Design](06_Specialisation_Website_Design/) | HTML/CSS basics, page structure | Practical assessment |
| [07_Project](07_Project/) | Integrated project: plan, create, present | Project submission |

## Notes

Students choose a specialisation track (Digital Printing or Website Design) alongside the shared Graphics Practical module. The Project module brings all tracks together.
