**Type:** 📖 Lesson
**Grade:** Grade 9
**Module:** Specialisation: Digital Printing
**Semester:** Semester 2
**Sequence:** 21 of grade year
**Status:** 📋 Šablonas

## Overview

1️⃣ Completion Definition\
Student applies margins and grid logic to create a clean page structure and can explain how grid constraints improve readability and alignment.

---

## 1️⃣ Completion Definition

Student can explain page layout principles (margins, grid logic).

## 2️⃣ Minimum Evidence

Annotated layout sketch or digital draft showing margins + grid columns/rows, with a short explanation of two alignment decisions.

## 3️⃣ Bloom's Taxonomy Level

Apply → Explain

## 4️⃣ Readiness Gate

Student can align elements consistently to a grid (not “eyeballing”) and can identify misalignment errors.
