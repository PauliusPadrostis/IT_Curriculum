**Grade:** Grade 9

## Overview

- Format identification
- Resolution reasoning
- Correct misuse cases.

---

## 1️⃣ Completion Definition

Student demonstrates independent competence in basic graphics foundations: choosing raster/vector appropriately, handling resolution, and producing an output that meets a technical specification.

## 2️⃣ Minimum Evidence

Assessment includes:\
one classification/justification item (raster vs vector + format)\
one applied edit task (resize/crop/export)\
one output submission meeting fixed constraints (dimensions + format).

## 3️⃣ Bloom's Taxonomy Level

Apply → Analyze

## 4️⃣ Readiness Gate

Output meets specification exactly (format + size) and student can justify choices.
