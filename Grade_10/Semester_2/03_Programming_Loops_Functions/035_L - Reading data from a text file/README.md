# L35 — Reading data from a text file

**Type:** 📖 Lesson
**Grade:** Grade 10
**Module:** Programming – Loops, Algorithms & Functions
**Semester:** Semester 2
**Sequence:** 35 of grade year
**Parent topic:** File I/O Introduction
**Status:** 🚧 WIP

---

## Overview

- What external data means (file vs console input)
- Opening and reading a simple .txt file line by line in C++
- Basic ifstream usage: open, read lines in a loop, close
- Printing file contents to console

## Sub-topics

- File as a data source (why not just cin?)
- ifstream basics: declaration, open(), is_open() check
- Reading lines with getline() in a while loop
- Closing the file

---

## 1️⃣ Completion Definition

Student can write a C++ program that opens a text file, reads its contents line by line, and prints them to the console.

## 2️⃣ Minimum Evidence

Working program that reads and displays all lines from a provided .txt file without errors.

## 3️⃣ Bloom's Taxonomy Level

Level 2 — Understand / Apply

## 4️⃣ Readiness Gate

Student understands while loops and string variables from prior lessons.

## 5️⃣ Common Failure Modes

- Forgetting to check if file opened successfully
- Confusion between cin and file stream
- Not understanding that the file must exist in the right directory
