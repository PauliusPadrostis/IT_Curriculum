# I36 — File reading mini-task

**Type:** 🔗 Integration practice
**Grade:** Grade 10
**Module:** Programming – Loops, Algorithms & Functions
**Semester:** Semester 2
**Sequence:** 36 of grade year
**Parent topic:** File I/O Introduction
**Status:** 🚧 WIP

---

## Overview

- Students read numeric data from a text file and apply a learned algorithm pattern (sum, count, average, min, or max) to the file data
- Combines file I/O from lesson 035 with algorithm patterns from lessons 023–026

## Sub-topics

- Reading numbers from a file (converting string lines to integers)
- Applying accumulation patterns to file data
- Comparing file-based vs console-based program structure

---

## 1️⃣ Completion Definition

Student can read numeric data from a text file and compute a result (sum, average, min, max, or count) using a loop.

## 2️⃣ Minimum Evidence

Working program that reads a file of numbers and produces correct computed output.

## 3️⃣ Bloom's Taxonomy Level

Level 3 — Apply

## 4️⃣ Readiness Gate

Completed lesson 035 (file reading basics) and lessons 023–026 (algorithm patterns).

## 5️⃣ Common Failure Modes

- String-to-int conversion errors
- Off-by-one errors when combining file reading loop with accumulation
- Attempting to read past end of file
