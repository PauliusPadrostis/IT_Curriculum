# Grade 10 – Semester 1

**Semester goal:** Build parallel competency in data handling (spreadsheets) and first programming (Python), so students can work with structured information both manually and programmatically.

## Modules

| Module | Topic | Assessment |
|--------|-------|------------|
| [01_Data](01_Data/) | Spreadsheet data: collection, cleaning, joins, pivot tables | Practical assessment |
| [02_Programming](02_Programming/) | Python basics: input/output, variables, types, conditions | Practical assessment |

## Notes

Both modules run concurrently. Data skills reinforce the idea of structured information; Programming introduces the same structures computationally. Co-schedule the practical assessments at the end of semester.
