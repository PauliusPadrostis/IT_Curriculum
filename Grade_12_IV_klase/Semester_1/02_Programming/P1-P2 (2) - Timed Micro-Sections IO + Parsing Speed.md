# P1–P2 (2) — Timed Micro-Sections: I/O + Parsing Speed

**Type:** 🏃 Practice
**Grade:** Grade 12 (IV gimnazijos klasė)
**Module:** Programming – File I/O, Parsing & Records
**Semester:** Semester 1
**Status:** 🚧 WIP

---

## Context

Short tasks purely on reading + parsing + exact output.

## Completion Definition

Student independently completes short, timed programming sections that require:

- Correct interpretation of a specification

- Accurate file reading and parsing

- Correct data-type conversion

- Producing required formatted output

Execution must be both **correct and within defined time constraints**.

Focus is on eliminating hesitation and formatting errors under pressure.

---

## Minimum Evidence

Student submits within time limit:

- Working program for each micro-section

- Correctly formatted output

- Recorded completion time (teacher logged)

Each section must:

- Parse file correctly

- Produce exact output format

- Run without runtime error

No step-by-step scaffold provided.

---

## Cognitive Level (Bloom's)

**Bloom: Apply**

Primary goal is fluency under time constraint, not conceptual expansion.

---

## Readiness Gate

✔ All must be true:

- Program runs without error

- Parsing correct

- Output formatting exact

- Completion within defined time threshold

- No debug output

If time exceeded or format incorrect → Gate not met.

---

## Common Failure Modes

- Rushing spec reading → wrong delimiter

- Minor formatting mismatch (extra space/newline)

- Forgetting type conversion

- Crashing on empty line

- Printing debug text

- Spending too much time over-optimizing trivial parts
