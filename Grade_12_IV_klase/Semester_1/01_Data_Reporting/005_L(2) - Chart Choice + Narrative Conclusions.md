## Context

**Exam link:** charts + “išvadų formulavimas ir pagrindimas” are explicit.

## Sub-topics

- Chart selection logic (what question → what chart)

- Writing conclusions grounded in computed evidence (not vibes)

- “So what?”: linking patterns → claim → justification

## Completion Definition

Student independently selects an appropriate chart type for a given reporting question, creates the chart correctly, and writes a concise conclusion that:

- States the main finding supported by the chart

- References at least one specific quantitative value (number, %, difference)

- Avoids claims not supported by the displayed data

Chart must be correctly configured (axes, labels, units, scale, series) and readable for assessment submission.

---

## Minimum Evidence

Student submits:

- The chart (embedded in report or exported)

- The underlying data range used

- A written conclusion (4–6 sentences) including:

  - one key trend or comparison

  - one numeric reference

  - one limitation or caution (e.g., missing data, outliers, category imbalance)

No sentence starters. No guided scaffold.

---

## Cognitive Level (Bloom's)

**Bloom: Analyze → Evaluate**

- Analyze which visualization matches the question/data type

- Evaluate what claim is justified and communicate it precisely

(Within Grade 12 exam-appropriate reporting expectations.)

---

## Readiness Gate

✔ All must be true:

- Chart type matches task intent (trend / comparison / composition / distribution)

- Chart has correct labels + units + readable scale

- Data range is correct (no missing or extra rows)

- Written conclusion contains:

  - ≥1 numeric reference

  - no unsupported inference

If any condition fails → Gate not met.

---

## Common Failure Modes

- Choosing a visually attractive chart rather than correct chart type

- Misleading scale (truncated axis without justification)

- Missing labels/units making interpretation impossible

- Wrong data range (headers included as data, partial selection)

- Conclusion restates chart title but gives no quantified finding

- Overclaiming causation from correlation

- Ignoring outliers or missing data effects
