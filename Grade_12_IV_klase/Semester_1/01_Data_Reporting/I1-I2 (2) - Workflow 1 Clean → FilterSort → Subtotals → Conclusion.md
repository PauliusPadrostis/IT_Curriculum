# I1–I2 (2) – Workflow 1: Clean → Filter/Sort → Subtotals → Conclusion

**Type:** 🔗 Integration
**Grade:** Grade 12 (IV gimnazijos klasė)
**Module:** Data – Reporting & Analysis
**Semester:** Semester 1
**Status:** 🚧 WIP

---

## Context

**Integration target**

- Students run the full pipeline on one dataset with strict correctness checks.

## Completion Definition

Student independently executes a full reporting workflow on a flawed dataset:

1. Cleans structural issues

2. Applies correct multi-criteria filter/sort

3. Generates grouped report with subtotals

4. Writes a concise numeric conclusion

All steps must be structurally coherent and logically connected.

---

## Minimum Evidence

Student submits:

- Cleaned dataset

- Filter/sort configuration visible

- Grouped report with correct subtotals

- 3–5 sentence written conclusion including ≥1 numeric reference

No guided checklist. No scaffold.

---

## Cognitive Level (Bloom's)

**Bloom: Apply → Analyze → Evaluate**

Sequential execution + justification of findings.

Appropriate for Grade 12 integration phase.

---

## Readiness Gate

✔ All must be true:

- Dataset passes hygiene criteria

- Filter/sort matches specification exactly

- Subtotals mathematically correct

- Conclusion references correct numeric result

- No step contradicts previous step

Any break in workflow logic → Gate not met.

---

## Common Failure Modes

- Cleaning only formatting, not structure

- Filtering before cleaning → distorted counts

- Incorrect grouping hierarchy

- Subtotal mismatch vs grand total

- Writing conclusion before verifying numbers

- Logical disconnect between filter and conclusion
