# P11 (1) — Timed Execution Rehearsal (Data Practical Simulation)

**Type:** 🏃 Practice
**Grade:** Grade 12 (IV gimnazijos klasė)
**Module:** Data – Reporting & Analysis
**Semester:** Semester 1
**Sequence:** 11 of grade year
**Lessons allocated:** 1
**Status:** 🚧 WIP

---

## Context

Hard time limit, strict output format, error penalties for mis-aggregation.

## Completion Definition

Student independently completes a full data practical simulation under strict timed conditions, including:

- Structural cleaning

- Correct tool selection (filter/sort, subtotals, or pivot as required)

- Chart production if specified

- Short numeric-based conclusion

Execution must demonstrate:

- Procedural fluency

- Correct sequencing

- Time control

---

## Minimum Evidence

Student submits within defined time window:

- Final cleaned dataset

- Required report output(s)

- Required chart(s) if applicable

- Written conclusions

- Recorded completion time

No scaffolding. No mid-task correction.

---

## Cognitive Level (Bloom's)

**Bloom: Apply → Analyze (Fluency Conditioning Level)**

Focus is execution accuracy under time constraint, not conceptual expansion.

---

## Readiness Gate

✔ All must be true:

- Completed within time limit

- No structural dataset errors

- Correct method selection

- All numeric outputs verifiable

- Written conclusion includes ≥1 numeric reference

If time exceeded or structural error present → Gate not met.

---

## Common Failure Modes

- Time misallocation (over-cleaning or over-formatting)

- Panic tool switching mid-task

- Forgetting refresh after pivot

- Arithmetic mismatch in subtotals

- Writing conclusion without checking numbers
