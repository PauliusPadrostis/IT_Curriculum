## Completion Definition

Student independently completes timed Data-practical sections that:

- Require structured dataset cleaning and/or transformation

- Apply filtering, grouping, or pivot logic as specified

- Produce correctly formatted outputs (tables + conclusions if required)

- Increase in structural difficulty across the 3 sessions

Student demonstrates:

- Accuracy under time constraint

- Method selection discipline (subtotals vs pivot)

- Formatting precision

- Written conclusion clarity (if required)

---

## Minimum Evidence

For each timed section, student submits:

- Completed spreadsheet/file

- Final outputs (tables + charts if required)

- Written conclusion (if required)

- Logged completion time

Requirements:

- No guided steps

- No scaffold prompts

- Increasing structural difficulty across T1 → T3

---

## Cognitive Level (Bloom's)

**Bloom: Apply → Analyze**

- Apply workflow mechanics

- Analyze dataset to choose correct aggregation method

Aligned with VBE 2 data practical conditioning.

---

## Readiness Gate

✔ All must be true:

- Correct data cleaning (if required)

- Correct method selection (subtotals/pivot as appropriate)

- Correct numerical results

- Correct formatting

- Completed within defined time threshold

If any workflow component fails or time exceeded → Gate not met.

---

## Common Failure Modes

- Choosing wrong reporting method

- Mis-grouping fields in pivot

- Sorting without grouping correctly

- Incorrect subtotal ranges

- Formatting drift under time pressure

- Weak or unsupported written conclusions

- Running out of time due to poor sequencing
